# Security Policy

This repository is a public Android learning project.

## Reporting

Please do not post private credentials or device data in a public issue. Open a short issue first, then share details privately with the maintainer.

## Project rules

- Do not commit debug APK files.
- Do not commit signing material or local machine configuration.
- Build public artifacts from the release variant.
- Keep sensitive Android permissions limited to features that need them.
- Review deep links before adding new routes.
