## 本次修复的问题

### 1. FilterChip 文字消失问题 ✅
**问题描述**: 选中 FilterChip 后，文字和图标消失不可见
**根本原因**: FilterChip 的颜色配置不完整，选中状态下缺少 `selectedLeadingIconColor` 和 `selectedTrailingIconColor`
**修复方案**: 
- 在 `TaskDetailScreen.kt` 的 `chipColors()` 函数中添加了完整的颜色配置
- 添加了 `iconColor`, `selectedLeadingIconColor`, `selectedTrailingIconColor`

**验证点**:
- [ ] 编辑任务时，分类 FilterChip 选中后文字可见
- [ ] 编辑任务时，分类 FilterChip 选中后图标可见
- [ ] 编辑任务时，优先级 FilterChip 选中后文字可见

---

### 2. 关联任务无法添加 ✅
**问题描述**: 点击"添加关联"后，搜索框立即消失，无法输入搜索内容
**根本原因**: `showLinkSearch` 和 `linkSearchQuery` 状态变量定义在 `if (taskId != -1L)` 块内部，每次重组都会重置为初始值
**修复方案**: 将这两个状态变量移到 Composable 函数顶层（第 51-52 行），使其在整个函数生命周期内保持状态

**验证点**:
- [ ] 编辑已有任务时，点击"添加关联"按钮
- [ ] 搜索框应该正常显示并保持可见
- [ ] 可以输入搜索关键词
- [ ] 搜索结果正常显示
- [ ] 点击搜索结果可以成功创建关联

---

### 3. 设置页面按钮样式问题 ✅
**问题描述**: 导出备份按钮颜色太淡，文字不清晰
**根本原因**: Button 使用了默认的 Material 3 颜色，与浅色主题不够对比
**修复方案**: 
- 为"导出备份"按钮添加显式颜色：`containerColor = primary`, `contentColor = White`
- 为"导入备份"按钮添加显式颜色：`contentColor = primary`
- 两个按钮都添加了 `fontWeight = FontWeight.SemiBold`

**验证点**:
- [ ] 设置页面"导出备份"按钮颜色清晰可见
- [ ] 按钮文字"导出备份"清晰可读
- [ ] "导入备份"按钮边框和文字颜色正常

---

### 4. 排序方式 FilterChip 样式统一 ✅
**问题描述**: 设置页面的排序方式 FilterChip 样式与任务编辑页面不一致
**根本原因**: 设置页面的 FilterChip 没有使用显式的颜色配置
**修复方案**: 为排序方式的 FilterChip 添加了与任务编辑页面相同的颜色配置

**验证点**:
- [ ] 设置页面排序方式 FilterChip 选中后文字可见
- [ ] 选中状态颜色为 primary 色，文字为白色
